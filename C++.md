## Conditionals and Loops ##

### Always Wrap Body in Braces ###
```cpp
// Good
if( condition ){
    doSomething();
}

// Bad
if( condition )
    doSomething();

// Bad
if( condition ) doSomething();
```

### Body Gets Own Lines ###
```cpp
// Good
while( condition ){
    doSomething();
}

// Bad
while( condition ){ doSomething(); }
```

### Open Brace On Keyword Line ###
```cpp
// Good
for( condition ){
    doSomething();
}

// Bad
for( condition )
{
    doSomething();
}
```

### Closing Brace Gets Own Line ###
```cpp
// Good
if( condition ){
    doSomething();
}
else {
    doSomethingElse();
}

// Bad
if( condition ){
    doSomething();
} else {
    doSomethingElse();
}

// Bad
if( condition )
{
    doSomething();
} else
{
    doSomethingElse();
}
```
